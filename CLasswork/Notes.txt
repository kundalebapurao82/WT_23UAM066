Web Technology:
Web technology refers to the tools, protocols, and languages (like HTML, CSS, JavaScript, servers, databases) used to build, run, and interact with websites and web applications, enabling communication and content delivery over the internet

- used for activities from browsing to online shopping and banking

ex:
Viewing a Website: HTML structures the text, CSS styles it, and JavaScript makes a button work when clicked.
Online Banking: Uses frontend (login page) and backend (database for accounts, security) technologies.
Streaming Video: Involves server-side delivery, browser playback, and interactive controls.

###### Events
Events is an a action that happens in a browser.
e.g. clicking button, moving mouse, pressing key, submitting form, loading page, etc.


JS can detect these events and respond to them.

Types of events:
- Mouse event
	click, dblclick, mouse hover, mouse out, 
- Keyboard event
	- key down, key up
- Browser event
	- load, resize, scroll





# React
Angular is Framework
React is Library

Angular is tyscript based frontend framework developed by Google.
It helps us to build a single page application (SPA), where the page does not fully reload while navigation between different views.
tools used for angular:
Node.js is a runtime environment that allows us to run JavaScript code outside the browser, often used for server-side development.
NPM (Node Package Manager) is a package manager for JavaScript, used to install and manage libraries and dependencies in Node.js projects.
Angular CLI (Command Line Interface): To create or manage angular apps/projects.

// install node js

node --version
npm install -g @angular/cli
ng new myProject

to run angular project:
ng serve or ng s

PSA includes:
components: building blocks of UI eg, home, about, contact us, etc

Package.json : Includes all the dependencies and scripts for the project. 
Dependancies used for production . 
DevDependencies used for development only.

Angular.json : configuration file for angular project. 
STyle.css : global styles for the project
.gitignore: specifies files and directories that should be ignored by Git version control, preventing them from being tracked or included in commits. 
tsconfig.json: typescript configuration file
index.html: main HTML file, runs initially when the app loads. Loads angular .
app.config.ts:app configuration file, used to define routes and other settings for the application.
main.ts: entry point of the application.

================= Services ==================
Serivces is like a helper class, it is used to store data and logic so many components can use it.
Instead of writing code in multiple codes, we write it once in a service and use it in multiple components.


to create service: ng g service serviceName
it created 2 files: serviceName.service.ts and serviceName.service.spec.ts
service file is used to write logic and data, spec file is used for testing.
