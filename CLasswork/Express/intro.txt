Routing in Express.js
Routing refers to determining how an application responds to a client request for specific endpoint(url) and HTTP methods.

Components of routing:
1. URL path
2. HTTP methods
3. Handler function

Create 3-4 routes:
1. app.get('/home', (req, res) => {
     res.send('Welcome to the Home Page');
    });
2. app.get('/resources', (req, res) => {
     res.send('This is the resources page');
    });
3.app.get('/about', (req, res) => {
     res.send('This is the about page');
    });
4. app.get('/contact', (req, res))=>{
        res.send('Contacts page');
    }
5. app.get('/services', (req, res) => {
     res.send('This is the services page');
    });

    how to run this code:
1. Save the code in a file named app.js
2. Open terminal and navigate to the directory where app.js is located
3. Run the command: node app.js
4. Open a web browser and navigate to http://localhost:3000/home, http://localhost:3000/resources, http://localhost:3000/about, http://localhost:3000/contact, and http://localhost:3000/services to see the respective responses.
