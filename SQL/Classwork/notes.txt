======== WEB SOCKETS ========
ORM: Object Relational Mapping
 - Instead of writing SQL queries we use JS objects to interact with the database.
 without ORM: SELECT * FROM Stdudent
 with ORM: Student.findAll()

Websocket:
 - allows us real-time communication between client and server
 - HHTP -> request and response
 - websocket -> continuous connection between client and server

 Eg. chat applications like whatsapp, live stock price updates, online gaming, notification.

To install websocket in our project:
